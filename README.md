# Тестовый проект автоматизированного тестирования API

## Технологический стек
- **Java 17**
- **JUnit 5**
- **RestAssured**
- **WireMock**
- **Allure**
- **Maven**

## Запуск тестов
Для запуска всех тестов и генерации результатов Allure:
```bash
mvn clean test
```

## Отчетность
Для генерации и просмотра интерактивного отчета Allure:
```bash
mvn allure:serve
```

Отчет включает:
- Подробные шаги выполнения каждого теста.
- Логи HTTP запросов и ответов (через фильтры RestAssured).
- Статистику и тренды прохождения тестов.

## Тестовое покрытие
- **Аутентификация (LOGIN)**: Проверка авторизации по токену и обработка ошибок.
- **Бизнес-действия (ACTION)**: Гарантия того, что действия доступны только для авторизованных сессий.
- **Управление сессиями (LOGOUT)**: Проверка завершения сессии и безопасности после выхода.
- **Безопасность**: Валидация защиты всех эндпоинтов с помощью API ключа (X-Api-Key).
